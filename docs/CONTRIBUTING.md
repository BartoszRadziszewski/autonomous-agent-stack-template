# Zasady współtworzenia projektu

Witaj Studencie/Partnerze! Twoje wsparcie rozwija standard Digital Finance Excellence (https://dfe.org.pl)

## Jak dodać nowego skilla?
1. Stwórz plik `skill_nazwa.md` w `.claude/skills/`.
2. Opisz placeholdery i logikę działania.
3. Przetestuj w swoim lokalnym środowisku.
4. Zgłoś Pull Request do głównego repozytorium.

## Standardy
- Dokumentacja w Markdown.
- Kod zgodny z PEP 8.
- Opis zmian w `CHANGELOG.md`.
